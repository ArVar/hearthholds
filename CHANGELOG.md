# Changelog

Notable changes to Hearthholds are documented here. The project follows
[Semantic Versioning](https://semver.org/) while the file format and user
experience remain in alpha.

## 0.1.0-alpha.1 – 2026-07-18

First Friends & Family alpha:

- Local-first settlement editor with bundled and blank templates.
- Buildings, terrain, infrastructure, construction phases, and presentation mode.
- Population, workforce, resources, cycle production, and treasury management.
- Versioned JSON import/export and automatic browser storage.
- Installable offline PWA with controlled updates.
- German and English interfaces with light and dark themes.
- Automated Chromium checks and GitHub Pages deployment.

Known limitations include local-device-only storage, desktop-first interaction,
no complete multi-document backup, and no automated Firefox or WebKit coverage.
