# Privacy

Hearthholds is a local-first application. It has no user accounts, backend,
cloud synchronization, analytics, advertising, or telemetry.

## Data stored on your device

Settlements and templates are stored in the browser's IndexedDB. Language and
appearance preferences use local storage. The offline application files use
the browser's Cache Storage. A JSON export is created only when you explicitly
request one.

This data belongs to the browser profile and device on which it was created.
Clearing site data, resetting the browser profile, or uninstalling the app
while also removing its data can permanently delete it. Export important
settlements regularly.

## Network and public reports

When Hearthholds is hosted on GitHub Pages, GitHub processes the network
requests needed to serve the application under its own privacy terms.
Hearthholds itself does not send settlement contents to GitHub or another
service.

GitHub issues and their attachments are public. Remove names, maps, notes, and
other confidential campaign information before sharing a screenshot or JSON
file. Security-sensitive reports should follow [SECURITY.md](SECURITY.md).
