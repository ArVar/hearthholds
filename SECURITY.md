# Security Policy

Hearthholds is currently a Friends & Family alpha. Security fixes target the
latest published alpha; older builds are not maintained separately.

## Reporting a vulnerability

Please use the repository's **Security** tab and choose **Report a
vulnerability** to create a private security advisory. If private reporting is
not available yet, contact the maintainer through the channel used for your
test invitation.

Do not disclose vulnerability details, private settlement exports, or campaign
data in a public issue. A report should include the affected version, impact,
and minimal reproduction steps. You can expect an acknowledgement within seven
days, although this alpha does not yet have a formal response SLA.
