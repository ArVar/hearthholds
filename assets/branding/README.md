# Hearthholds branding

Die Dateien in diesem Ordner bilden den freigegebenen Markenstand ab. Das `H`
der quadratischen Icons wurde direkt aus dem Master-Schriftzug übernommen und
ist keine separat erzeugte Variante.

| Datei | Verwendung |
| --- | --- |
| `hearthholds-wordmark-master.png` | Master-Logo, 2400 × 1000 px |
| `hearthholds-wordmark.webp` | Weboptimiertes Logo, 1200 × 500 px |
| `hearthholds-icon-master.png` | Quadratisches Icon-Master, 1024 × 1024 px |
| `hearthholds-icon-{size}.png` | Browser-, Apple- und App-Icons in Standardgrößen |
| `hearthholds-icon-64.webp` | Kompaktes Markenicon für die laufende Oberfläche |
| `hearthholds-favicon.ico` | Mehrgrößen-Favicon für klassische Browser |
| `hearthholds-wood-texture.webp` | Freigegebene Holztextur für markante GUI-Flächen |
| `hearthholds-wood-texture-subtle.webp` | Zurückhaltende Variante für größere GUI-Flächen |

Bedienfunktionen verwenden weiterhin die bestehenden, selbsterklärenden
Lucide-Icons. Die Branding-Assets ersetzen keine funktionalen UI-Symbole.

## Lizenz und Markennutzung

Die beiden Holztexturen sind als Projekt-Artworks unter `CC BY-SA 4.0`
freigegeben. Name, Schriftzüge, App-Icons und Favicon sind reservierte
Markenassets. Sie dürfen unverändert als Teil einer unveränderten offiziellen
Hearthholds-Version weitergegeben und zur zutreffenden Bezugnahme auf das
Projekt gezeigt werden. Veränderte Distributionen benötigen einen eigenen
Namen und eine eigene visuelle Identität.

Die verbindliche Abgrenzung steht in [LICENSING.md](../../LICENSING.md).
